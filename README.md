# playa2

A comfy command line audio player.

## Build

```
./do.sh config && ./do.sh build
```

## Run

```
./do.sh run -f /path/to/playlist.txt
```

`playlist.txt` is a file containing paths to the audio files to play, one per line.

## Usage

* 1-0: set track
* q-p: set track progress
* j: down
* k: up
* space: play/pause
